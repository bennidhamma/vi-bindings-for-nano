# vi-bindings-for-nano
Make nano usable

Run these simple commands and your nano will have vi-bindings in no time!

Step 1: 
-------
```
$ apt-get install vim
$ echo 'alias nano=vim' >> ~/.bashrc
$ . ~/.bashrc
```

Step 2:
-------
```
$ nano /path/to/your/file
```

Step 3:
------
Profit!
